Mini Project 25: Event Countdown Manager

📌 Description:
A Python-based Event Countdown Manager that lets users add upcoming events with their dates,
store them in a JSON file, and view a countdown of days left until each event using the datetime module.

🛠️ Features:
- Add events with name and date (YYYY-MM-DD)
- View all upcoming events with countdown (days left)
- Stores events persistently in a JSON file
- Uses datetime for countdown calculations
- Menu-driven interface with Add, View, and Exit options

📂 Files Used:
- events.json – stores the event data persistently in JSON format

📅 Technologies:
- Python's datetime module
- JSON for file handling
- Simple menu loop using match-case
