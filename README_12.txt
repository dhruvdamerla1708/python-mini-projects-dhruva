Mini Project 12: Common Course Finder

Description:
-------------
This project compares the course lists of two students and performs set operations to find:
- Common courses
- Courses unique to each student
- Whether their lists are disjoint

Features:
---------
- Takes comma-separated input for both students
- Uses set operations: intersection, difference, isdisjoint
- Displays a clear comparison of their course selections

Usage:
------
Run the script and input two sets of comma-separated course names.
