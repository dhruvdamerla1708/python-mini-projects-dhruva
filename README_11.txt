Mini Project 11: Unique Word Analyzer

Description:
-------------
This project takes a paragraph input from the user and analyzes the total number of words, 
the number of unique words, and displays the unique words in a sorted manner.

Features:
---------
- Splits the paragraph into words
- Removes duplicates using set
- Displays total and unique word counts
- Outputs sorted list of unique words

Usage:
------
Run the script and enter a paragraph when prompted.
