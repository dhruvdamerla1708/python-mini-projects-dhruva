# 🎓 Student Marks CSV Manager

This mini project is a simple CSV-based student record manager that allows users to:

- Add new student marks
- View all records
- Search a student by name
- View students who scored more than a chosen mark

---

## 🛠 Features

- **Add Student**: Input name, subject, and marks.
- **View All**: Display all students from the CSV.
- **Search by Name**: Case-insensitive name search.
- **More than Marks**: Filter students scoring above a given mark.

---

## 📂 File Used

- `student.csv` — stores student data in `[name, subject, marks]` format.

---

## ▶️ How to Run

Run the script in a Python environment:

```bash
python student_csv_manager.py
```

Follow the menu options for operations.

---

## ✅ Requirements

- Python 3.x
- No external libraries needed (uses built-in `csv` module)
